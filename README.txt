2. La realización de esta aplicación web de ventas para docentes constará de varios componentes diferentes. Algunos de estos componentes se programarán, mientras que otros serán
implementaciones de programas de código abierto. El lenguaje utilizado estará dictado por su propósito. Las interfaces administrativas y de usuario utilizarán JSP para mostrar las páginas
y SQL para recuperar, insertar, eliminar y actualizar la base de datos. Se utilizará Visual Studio para hacer la aplicación web y enviar comandos a SQL para la parte
automatizada del proyecto como actualizar la información del usuario o productos. También se configurará con dos niveles de usuario. El primero que solo podrá ofrecer o comprar productos
El segundo tipo de usuario es el administrador. Tiene la capacidad de cambiar información en la base de datos. Este nivel de usuario solo estará disponible para los integrantes del equipo.


3. Las siguientes carpetas deben estar incluidas para el correcto funcionamiento o entendimiento del proyecto:

- SQL: Incluye todos los scripts para el funcionamiento de la base de datos. Todo hecho en SQL SERVER.

- BackEnd: Incluye la solución de Visual Studio con el proyecto del BackEnd.

4. Instrucciones de instalacion

- Descargue todas las carpetas que se encuentran en este repositorio.

- Ejecute el script de la creación de tablas que se encuentra en la carpeta SQL.

- Abra la solución de Visual Studio y corra el programa.
